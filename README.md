# 🚀 Acelerando a Aprendizagem com IA usando NotebookLM

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como desafio prático da DIO com o objetivo de explorar o potencial do NotebookLM como ferramenta de aprendizagem ativa utilizando Inteligência Artificial.

O tema escolhido foi:

👉 IA aplicada ao Marketing Digital, Automação e Produtividade.

A proposta foi utilizar o NotebookLM para organizar conteúdos, acelerar estudos, gerar resumos inteligentes, estruturar conhecimento e testar técnicas de engenharia de prompts aplicadas ao aprendizado moderno.

Além disso, o projeto busca demonstrar como a IA pode ser utilizada como ferramenta estratégica para produtividade, estudos e desenvolvimento profissional.

---

# 🎯 Objetivos de Estudo

- Aprender como utilizar o NotebookLM na prática;
- Explorar IA aplicada à organização de conhecimento;
- Melhorar produtividade nos estudos;
- Criar resumos inteligentes utilizando IA;
- Aplicar técnicas de Prompt Engineering;
- Desenvolver pensamento crítico usando IA;
- Estruturar conteúdos de forma estratégica;
- Criar materiais reutilizáveis para futuras revisões.

---

# 🧠 O que é o NotebookLM?

O NotebookLM é uma ferramenta de Inteligência Artificial desenvolvida pelo Google que permite criar cadernos inteligentes utilizando documentos, PDFs, links e textos como base de conhecimento.

A IA consegue:
- gerar resumos;
- responder perguntas;
- criar explicações;
- estruturar conteúdos;
- organizar conhecimento;
- auxiliar nos estudos;
- acelerar revisões.

## 🧪 Engenharia de Prompts

## 🔹 Prompt 1

```text
Crie um resumo estratégico sobre como a IA pode melhorar campanhas de marketing digital.
```

### ✅ Resultado Obtido
- Resumos rápidos;
- Organização eficiente;
- Insights relevantes;
- Melhor entendimento sobre automação.

### 🧠 Resposta Obtida (Síntese)
A IA explicou que ferramentas inteligentes podem automatizar processos, melhorar análises de campanhas e otimizar estratégias de marketing digital através de dados e produtividade.

### ⚠️ Dificuldade Encontrada
A resposta inicial ficou muito ampla e genérica.

### 💡 Lição Aprendida
Prompts mais específicos geram respostas mais precisas e úteis.

---


### ✅ Resultado Obtido

### 💡 Lição Aprendida

# ⚠️ Troubleshooting e Dificuldades Encontradas

Durante os testes realizados no NotebookLM, algumas respostas da IA ficaram:
- muito genéricas;
- superficiais;
- pouco específicas.

## ✅ Soluções Aplicadas

Para melhorar os resultados foi necessário:

- utilizar prompts mais detalhados;
- contextualizar melhor as perguntas;
- dividir perguntas complexas em partes menores;
- revisar informações geradas pela IA;
- ajustar a estrutura dos comandos.

Isso demonstrou como a engenharia de prompts influencia diretamente na qualidade das respostas.
